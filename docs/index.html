Installation

Cài đặt addon Genshoe trong Blender rất đơn giản.

  1. Tải file addon (.zip)
  2. Mở Blender
  3. Vào Edit → Preferences
  4. Chọn Add-ons
  5. Nhấn Install
  6. Chọn file .zip của addon
