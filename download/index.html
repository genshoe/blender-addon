Download Genshoe

Tải addon Genshoe cho Blender và bắt đầu thiết kế giày chuyên nghiệp.

Genshoe v1.0

Tương thích Blender 3.6+

Download (.zip)

File size: ~25MB

Hướng dẫn cài đặt

1. Mở Blender

2. Vào Edit → Preferences → Add-ons

3. Nhấn "Install"

4. Chọn file .zip vừa tải

5. Enable addon "Genshoe"

Những gì bạn nhận được

✔ UV to Flat Mesh Tool
✔ Reproject System
✔ Geometry Workflow
✔ Future Updates

Yêu cầu hệ thống

Blender: 3.6 trở lên

OS: Windows / macOS / Linux

RAM: 8GB+ khuyến nghị
